API Base:
GET /api/films — list semua film
GET /api/films/:id — detail film
POST /api/films — tambah film (auth required)
PUT /api/films/:id — edit film (auth required)
DELETE /api/films/:id — hapus film (auth required)
POST /api/auth/login — login
GET /proxy?url=... — proxy iframe (strip X-Frame-Options & CSP)
